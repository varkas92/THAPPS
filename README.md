# THAPPS

## Thermo-Hydraulic Analyser for Premise Plumbing Systems

THAPPS is a Python-based modelling framework for the thermo-hydraulic simulation and analysis of premise plumbing systems (PPS).

Current focus areas include:

- Hydraulic behaviour
- Water temperature dynamics
- Heat transfer and heat loss
- Water age analysis
- System performance assessment
